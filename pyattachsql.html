pyAttachSQL

pyAttachSQL is an Apache 2.0 licensed Python wrapper for libAttachSQL which is currently heavily in-development.

Project Information

The pyAttachSQL source code is hosted on GitHub. All code is automatically tested using Travis CI. The status of what is currently being worked on can be seen on our Waffle.io project board.

Documentation is in reStructuredText files included in the source and is compiled to generate our Read The Docs documentation.

Questions?

Feel free to file an issue, talk to us on #libAttachSQL on Freenode or email andrew@libattachsql.org.
