libAttachSQL

libAttachSQL is a lightweight C/C++ API for MySQL servers originally created by HP's Advanced Technology Group. It is desgined to be easy to use and easy to contribute to.

WARNING: THIS PROJECT IS IN A PRE-ALPHA STATE!

Project Information

The libAttachSQL source code is hosted on GitHub. All code is automatically tested using Travis CI with the build status shown above

Documentation is in reStructuredText files included in the source and is compiled to generate our Read The Docs documentation.

Questions?

Feel free to file an issue, talk to us on#libAttachSQL on Freenode or email andrew@libattachsql.org
