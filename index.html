libAttachSQL

libAttachSQL is a lightweight C API for MySQL servers originally created by HP's Advanced Technology Group. It is desgined to be easy to use and easy to contribute to.

WARNING: THIS PROJECT IS IN ALPHA, API MAY CHANGE!

News

Version 0.2.0 Alpha released!

2014-08-15

Hot on the tails of the first alpha release we have released the second alpha of libAttachSQL. This includes many fixes and enhancements including:

  • Improved memory handling
  • Buffered result set support
  • Asynchronous connect on first query
  • Documentation improvements including examples

For now this is still a source-only release and the source can be downloaded from GitHub.

Version 0.1.0 Alpha released!

2014-08-08

Our first early release version of libAttachSQL is out. It can currently connect to MySQL and perform basic queries. We do not recommend it for production at this time and the API will likely change a bit as we head towards a beta release.
You can download v0.1.0 directly from github.

Project Information

The libAttachSQL source code is hosted on GitHub. All code is automatically tested using Travis CI with the build status shown above

Documentation is in reStructuredText files included in the source and is compiled to generate our Read The Docs documentation.
