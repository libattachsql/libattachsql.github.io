Version 1.0.0 GA Released!

Tue 21 Oct 2014, 23:16

In News.

tags: releaseGA

Today marks the first GA release of libAttachSQL. Not a huge amount has changed over the RC release, almost exclusively bug fix changes to improve stability and error handling.

You can use the following links to download libAttachSQL 1.0.0:

  • Source Tarball
  • RHEL 6.x/CentOS 6.x x86_64: 6.x library RPM and 6.x development library RPM
  • RHEL 7.x/CentOS 7.x x86_64: 7.x library RPM, 7.x development library RPM and 7.x debuginfo RPM
  • PPA for Ubuntu 12.04 (Precise) and 14.04 (Trusty) (1.0.0 packages will be available in the next 24 hours)

More platforms will come soon.
